Development environment

For developing Hax data centers, the recommended tool is the Eclipse platform with the GroovyEclipse plugin installed. You will also need to download the HaxFramework Eclipse project from here, and add it to your Eclipse workspace. After that, see the README.md file of the HaxFramework project on how to proceed.
