Package: sk.hax.software.tutorial

[Groovy] Class CommandTaskTutorial

      • sk.hax.software.tutorial.CommandTaskTutorial

      • Abstract base class of a tutorial wrapper around any command task. Creates a child task instance on the background and guides the user through a number of steps. Each step displays a message and requires the user to enter specific input, e.g. a command. The steps are configured in subclasses by a simple object-based configuration.

          • protected java.util.List getSteps()

            Get the list of steps of the tutorial. Each step is a map of key-value pairs. Following keys are supported:

            • desc: description of the step, displayed to the user
            • fwd: required user input to pass the step and proceed to the next step
            • hndl: boolean value indicating whether after passing the step, the user input should be sent to the underlying task instance
            • bck: boolean value indicating whether the user is allowed to go back one step from the current step (by typing '<')
            • tnl: boolean value indicating whether tunneling mode should be started after passing the step, meaning that any subsequent user input will not be handled by the tutorial, until a null or escape character is received
            Returns:
            list of steps
