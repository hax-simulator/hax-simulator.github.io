      • putBootBlock

        void putBootBlock(java.net.URI uri)
        Put a boot block to this storage from the specified URI. The URI should point to a JAR file or a directory on the physical disk containing the boot task (kernel) and its dependencies.
        Parameters:
        uri - location of the boot task on the physical disk

      • putTaskBlock

        void putTaskBlock(java.lang.String mountPoint,
                java.net.URI uri)
        Put a task to this storage from the specified URI. The URI should point to a JAR file or a directory on the physical disk containing the task and its dependencies. The task is added to the specified mount point (a virtual path). Note that the path of the file on the virtual storage is exactly the mount point specified, including the file name.
        Parameters:
        mountPoint - virtual path to put the task to
        uri - location of the task on the physical disk

      • putDataBlock

        void putDataBlock(java.lang.String mountPoint,
                java.net.URI uri)
        Put data to this storage from the specified URI. The URI should point to a file or a directory on the physical disk containing the data. The data is added to the specified mount point (a virtual parent directory). Note that if the URI points to a file, the filename is added to the mount point on the virtual storage as is.
        Parameters:
        mountPoint - parent virtual directory to put the data to
        uri - location of the data on the physical disk
