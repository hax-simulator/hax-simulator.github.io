      • outbound

        void outbound(byte[] data,
            boolean synchronous)
        Send out outbound data and notify any registered listeners. The 'synchronous' parameter should be set to true set to true if the caller is expecting to receive some data in response to his outbound transmission and wants to process this data synchronously (in the same thread). In this case, the method of delivering the response data to the caller is up to the software in control of the device. Failure to set the parameter correctly may result in not being able to receive the expected response data at the required time.
        Parameters:
        data - the sent data
        synchronous - should be set to true for synchronous request-response communication scenarios
        Since:
        2.1_alpha
