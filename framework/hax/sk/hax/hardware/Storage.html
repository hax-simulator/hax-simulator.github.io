Interface Storage

  • All Superinterfaces:
    Device, java.util.Map<java.lang.String,java.lang.Object>

    public interface Storage
extends Device, java.util.Map<java.lang.String,java.lang.Object>
    A device representing a storage device. The storage is indexed by strings and stores arbitrary objects. Also, each stored object can be assigned a number of arbitrary attributes. A storage might be associated with a physical location to make it a permanent storage. The client must explicitly call the flush method to permanently store the contents of the storage.
