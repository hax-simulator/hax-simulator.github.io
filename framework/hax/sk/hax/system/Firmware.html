      • buildTask

        Task buildTask(java.lang.Object object)
        Create a task instance from an object. The object may be a string, byte array or class. In case of a string, it is interpreted as a script. In case of a byte array, it is interpreted as a JAR. In case of a class, an instance of the class is returned.
        Parameters:
        object - task object, may be either a string, byte array or class
        Returns:
        task instance for the task object, null if it cannot be constructed
