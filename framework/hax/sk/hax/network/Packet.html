Class Packet

  • java.lang.Object
    • sk.hax.network.Packet
  • Direct Known Subclasses:
    NetworkPacket

    public abstract class Packet
extends java.lang.Object
    Abstract representation of a chunk of data (packet). Can be initialized from or converted to a byte array. Plainly encapsulates the packet represented by a byte array in an object. Should be used as the base class for classes representing higher layers of the network protocol stack.
