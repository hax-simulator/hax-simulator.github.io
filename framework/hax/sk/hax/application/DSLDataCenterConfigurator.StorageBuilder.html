      • boot

        public void boot(java.lang.String path)
        Define the boot block of the storage. The specified path should point to the operating system for the machine. If the path points to a directory, a JAR is created from it and is used as the operating system. In this case, the kernel class must be defined as a fragment of the path. If the path points to a file, it must be a JAR file containing the operating system, where the kernel class is set as the main class using the JAR manifest.
        Parameters:
        path - reference to the operating system in form of an URI

        public void task(java.lang.String mountPoint,

        Define a task block of the storage. The task block is created directly on the specified mount point of the storage. If the path points to a directory, a JAR is created from it and is set to the specified mount point. In this case, the main class may be defined as a fragment of the path.       • data

        public void data(java.lang.String mountPoint,
        java.lang.String path)
        Define a data block of the storage. The data block is created as one or more children under the specified mount point of the storage. If the path points to a directory, the directory contents are recursively added to the storage as children of the mount point. The contained directory structure and file names are preserved. If the path points to a file, the file is added to the storage as a child of the mount point. The file name is preserved.
        Parameters:
        mountPoint - storage mount point
        path - reference to the data in form of an URI
