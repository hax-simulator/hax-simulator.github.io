Package: sk.hax.software.os.command

[Groovy] Class Telnet

        • sk.hax.software.os.command.Telnet

        • Client task for communicating with remote tasks published on the network. Does not implement any specific communication protocol, supports only raw data communication (using byte arrays). Operates in either unconnected or connected mode. In unconnected mode, provides a custom command interpreter allowing to specify a network address and port to connect to, or exit the task. In connected mode, forwards any data coming from the terminal to the connected network address and port, as well as any data received from the network back to the terminal.
