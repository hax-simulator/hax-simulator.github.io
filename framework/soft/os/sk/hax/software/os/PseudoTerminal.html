Package: sk.hax.software.os

[Groovy] Class PseudoTerminal

        • sk.hax.software.os.PseudoTerminal

        • Represents a virtual terminal device. Consists of a pair of virtual devices called slave and master. The slave emulates a real terminal device, but instead of interfacing with hardware, it interfaces with the master. All data sent to the master is forwarded to the slave, which acts as if it received incoming data from a real terminal. Likewise, all data published to the slave is forwarded to the master, triggering a callback. This mechanism is used by tasks which invoke some child task and need to intercept or manipulate the data flowing in and out of that child task. In this case, they simply create a pseudoterminal and assign the slave device to the child task in place of the real terminal. They are then able to subscribe to the master to receive all outgoing data from the slave, or send data using the master directly to the slave.
