Package: sk.hax.software.os

[Groovy] Class Shell

        • sk.hax.software.os.Shell

        • Implements the command interpreter for HaxOS. Interprets and executes commands entered using the terminal. Is able to start and execute other tasks. Is responsible for maintaining the working directory.
